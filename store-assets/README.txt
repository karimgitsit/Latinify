CHROME WEB STORE ASSET REQUIREMENTS
====================================

REQUIRED:
---------
1. promo-small.png (440 x 280 px)
   - Marketing tile shown in search results
   - Should include logo, name, and tagline
   - Recommended: Roman aesthetic, "Latinify" text, maybe a preview

2. screenshot-1.png (1280 x 800 px OR 640 x 400 px)
   - At least 1 required, up to 5 allowed
   - Show the extension in action
   - Ideas:
     * Before/after of a webpage
     * The popup UI
     * A translated Wikipedia page

OPTIONAL (but recommended):
---------------------------
3. promo-large.png (920 x 680 px)
   - Larger marketing tile
   - Used for featured placements

4. promo-marquee.png (1400 x 560 px)
   - Wide banner
   - Used at top of category pages

TIPS:
-----
- Use consistent branding (ivory, gold, charcoal colors)
- Include "Fac Latinum" tagline
- Show real examples of translated text
- Keep text readable at small sizes

COLOR PALETTE:
--------------
Ivory/Cream: #F5F2EB
Charcoal: #2C2C2C
Roman Gold: #C9A227
