Latinify Privacy Policy

Fac Latinum — Make it Latin

Summary: Latinify does not collect, store, or transmit any personal data. Your privacy is fully respected.

Overview

Latinify is a browser extension that translates webpage text into Latin. We are committed to protecting your privacy and being transparent about how the extension works.

What We Collect

Nothing. We do not collect, store, or transmit any personal data. We have no servers, no analytics, and no tracking of any kind.

How the Extension Works

When you activate Latinify on a webpage:

  • The extension reads the visible text on the page locally, in your browser
  • This text is sent to Google Cloud Translation API for translation
  • The translated text replaces the original on your screen
  • Original text is cached locally in your browser so you can restore it
  • No data is sent to our servers — we don't have any

Third-Party Services

We use Google Cloud Translation API to perform translations. When you translate a page, the text content is sent to Google's servers over a secure HTTPS connection.

Google's privacy policy applies to this data: https://policies.google.com/privacy

Local Storage

The extension stores the following data in your browser's local storage:

  • Original page text — to enable the restore functionality
  • Translation cache — to avoid redundant API calls and improve performance
  • Rate limiting counters — to prevent excessive API usage

This data never leaves your browser and is automatically cleared over time or when you clear your browser data.

Permissions Explained

  • activeTab — To read and modify text on the current page when you activate the extension
  • storage — To cache translations locally in your browser
  • scripting — To inject the translation script into web pages

Changes to This Policy

We may update this privacy policy occasionally. Any changes will be posted on this page with an updated revision date.

Contact

If you have any questions about this privacy policy or how Latinify works, please open an issue on our GitHub repository or contact us via the Chrome Web Store listing.
